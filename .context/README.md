# Immersive IA Traffic RRMX — AI Context

Esta carpeta contiene toda la información necesaria para que una IA
pueda retomar el trabajo en este mod sin perder contexto.

## Archivos de contexto

| Archivo | Contenido |
|---------|-----------|
| `project_overview.md` | Descripción del proyecto, objetivo, stack tecnológico |
| `mod_architecture.md` | Mapa de archivos del mod y su propósito |
| `design_decisions.md` | Decisiones de diseño tomadas y sus justificaciones |
| `dev_log.md` | Log de cambios por sesión de desarrollo |
| `base_reference.md` | Referencia de los valores BASE del juego para comparación |

## Cómo usar

1. Lee `project_overview.md` para entender el objetivo
2. Lee `mod_architecture.md` para entender qué hace cada archivo
3. Consulta `design_decisions.md` para saber el "por qué" de cada ajuste
4. Revisa `dev_log.md` para el historial de cambios

## Ramas

- `main` — versión publicada/estable
- `dev` — rama de desarrollo y pruebas activa
